#NATURE APP

### Installing

A step by step that tell you how to get this

Clone this repo

```
git clone https://github.com/MiguelMelo/nature-app-expo.git
```
Go to the directory

```
cd nature-app-expo
```

Install packages

```
npm install
```

Install expo-cli (if you don't have it)

```
npm install -g expo-cli
```

And finally... Run it!

```
npm start
```

*Everything* is going according to **plan**.
